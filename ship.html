Ship game

My ship game began as a personal project to test my very basic console knowledge. It had your ship fight against another, and if you won, you would gain upgrade points to improve your ship. After you upraded you went into another level and fought again. I called it a demo, because it had only 3 levels. it looks like this:

If you want to try the console demo, here are the install instructions.

Windows instructions:

  • Download the binary here.

  • Double click to start.

Macbook and other instructions.

  • copy the text from here. for mac users, command+a then command+c

  • go to cpp.sh then paste the text (Command+v) into the editor window. Remeber to remove the code already there.

  • click run. (note that cpp.sh may occasionialy break.)
