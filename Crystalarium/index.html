Crystalarium

Crystalarium is a game/software toy where you will be able to simulate 'rulesets' that can be used to make things like computers. Ideally, crystalarium will be a playground for making and messing with digital-like circuits and asorted weird stuff. Currently, much of the framework for making simulation systems is in place, and as of version 0.6, crystalarium comes with 3 rulesets, though you cannot make your own yet.

Below is a video that (poorly) shows off the latest features in the latest version of crystalarium. Primarily that the simulation is now more deterministic and the addition of another ruleset, wireworld.

For a description of the rules of each ruleset, along with a list of challenges to try, you can go to the wiki.
